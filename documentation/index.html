Documentation

We provide documentation for each versioned release and each 1.x and 2.x incremental build.

Latest 2.x Release (2.1.10)

The documentation for 2.x releases takes the form of API docs and guides.

The API docs and guides for the latest 2.x release (currently 2.1.10) are available at:

  • http://immutant.org/documentation/current/apidoc/

Latest 2.x Incremental Build

All successful incremental builds are published, but you must configure a repository to obtain them.

  • 2.x Incremental API docs

You can access the documentation for prior incremental builds on the 2.x builds page.

Latest 1.x Release (1.1.4)

The documentation for the latest 1.x release (currently 1.1.4) is available at:

  • http://immutant.org/documentation/1.1.4/ - Manual
  • http://immutant.org/documentation/1.1.4/ - Clojure API docs
  • Tutorials

You can access the documentation for prior releases on our releases page.

Latest 1.x Incremental Build

The documentation for the latest successful 1.x incremental build is always available at:

  • http://immutant.org/builds/LATEST/html-docs - Manual
  • http://immutant.org/builds/LATEST/html-docs/apidoc/ - Clojure API docs
  • Tutorials

You can access the documentation for prior incremental builds on our 1.x builds page.

Documentation Issues

If you find any issues with our documentation or have any questions about it, please let us know.
