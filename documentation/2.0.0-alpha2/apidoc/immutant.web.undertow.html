listen

(listen {:doc nil, :doc/format :markdown, :keys [configuration host port ssl-port ssl-context key-managers trust-managers], :or {:doc nil, :doc/format :markdown, host "localhost"}, :as options})

Return the passed listener-related options with an Undertow$Builder instance set accordingly, mapped to :configuration in the return value. If :ssl-port is non-nil, either :ssl-context or :key-managers should be set, too
